# CareSphere 🌿🩺

A holistic health & wellness dashboard powered by Python + SQL

CareSphere is an integrated health management system that unifies multiple aspects of personal wellness into one platform. It empowers users to track physical health metrics, plan nutrition, manage medications, log mental wellness, participate in fitness challenges, and monitor symptoms — all backed by a structured SQL database.

The vision of CareSphere is to provide a 360‑degree view of personal care and wellness, helping individuals build healthier habits, stay consistent, and gain actionable insights into their lifestyle.

## ✨ Key Features

- 📊 **Health Metrics Tracker** – log weight, steps, sleep, and calories with analytics
- 🥗 **Diet & Nutrition Planner** – create meal plans, calculate nutrients, and suggest smart substitutions
- 💊 **Medication Reminder** – schedule doses, track adherence, and flag missed medications
- 🌱 **Mental Wellness Journal** – record moods, activities, and analyze lifestyle correlations
- 🏃 **Fitness Challenge Tracker** – join challenges, track streaks, and compete on leaderboards
- 📑 **Reports & Insights** – generate personalized summaries and export health reports
- 🔔 **Notifications & Alerts** – unified reminders for medication, goals, and challenges

## 🧩 Tech Stack

- **Backend:** Python (Flask)
- **Database:** MySQL
- **Visualization:** Matplotlib / Plotly for charts and analytics
- **Optional Integrations:** APIs for fitness devices or nutrition databases
